ICASH Payment Collection API

Developed an API for ICASH Finance payment collection, enabling merchants to generate QRPh codes for client transactions. The API creates virtual accounts for each unique transaction, allowing clients to make payments via digital banking apps or e-wallets. Once a payment is verified, the system notifies the merchant of the successful transaction. It supports both static/dedicated virtual accounts per customer and dynamic virtual accounts for one-time use with reference numbers. Additionally, the API can send webhook or callback notifications to merchants' servers for real-time transaction updates.

C# ASP.Net Core IIS Windows Server REST API
